CSS and JavaScript animation performance

Animations are critical for user experience on many applications. There are many ways to implement web animations, such as  CSS transitions/animations  or  JavaScript based animations.

CSS Transitions

Provide an easy way to handle the changes DOM element properties as result of a interactions of its own properties or states.

Recommended to handle change of states on small elements.

#button {
  border-radius: 5px;
  display: inline-block;
  font-size: 10px;
  padding: 7px 15px;
  position: relative;
  background-color: transparent;
  border: 1px solid #39c2d7;
  color: #39c2d7;
  text-decoration: none;
}
#button:before {
  content: '';
  position: absolute;
  height: 100%;
  width: 0;
  left: 0;
  top: 0;
  z-index: -1;
  background: transparent;
  transition: .75s background ease, .75s width ease;
}
#button:hover {
  color: white;
}
#button:hover:before {
  background: #39c2d7;
  width: 100%;
}
