Transiciones con CSS

Usada para manipular propiedades entre dos estados.

Recomendada para cambios simples.

Hover

#button {
  border-radius: 5px;
  display: inline-block;
  font-size: 10px;
  padding: 7px 15px;
  position: relative;
  background-color: transparent;
  border: 1px solid #39c2d7;
  color: #39c2d7;
  text-decoration: none;
  z-index: 1;
}
#button:before {
  content: '';
  position: absolute;
  height: 100%;
  width: 0;
  left: 0;
  top: 0;
  z-index: -1;
  background: transparent;
  transition: .75s background ease, .75s width ease;
}
#button:hover {
  color: white;
}
#button:hover:before {
  background: #39c2d7;
  width: 100%;
}

Animaciones con CSS

Usada entre diferentes para manipular propiedades entre múltiples estados.

Puede controlarse el tiempo de ejecución así como de retardo.

Puede ejercutarse múltiples veces.

Uso de GPU.

Poco escalable.

Recomendado para animaciones simples.

@keyframes left {
    0% { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}

@keyframes right {
    0% { transform: rotate(0deg); }
  100% { transform: rotate(-360deg); }
}

#loader {
  height: 55px;
  width: 55px;
  position: absolute;
  left: 0;
  right: 0;
  top: 0;
  bottom: 0;
  margin: auto;
}

#loader:before, #loader:after {
  content: '';
  border-radius: 50%;
  display: block;
  width: 50px;
  height: 50px;
  position: absolute;
}
#loader:before {
  border-left: 1px solid red;
  animation: left 1s infinite;
  animation-timing-function: linear;
}
#loader:after {
  border-bottom: 1px solid blue;
  animation: right 1s infinite;
  animation-timing-function: linear;
}
